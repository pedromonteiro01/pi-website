DSD

Sistema de apoio à criação da Distribuição do Serviço Docente

A platform made to enance the process of building/making the DSD in a simple and quick way.

Description

In this project we are asked to do a platform that allows to perform a resource allocation of the teachers through the diverse disciplines in the university. This application should be designed to be a tool for the persons who will realise the allocation task.

Goals

Balanced Distribution

Allow the hours assigned to each teacher to be linearly distributed

Teachers History

Ability to see past data in order to have a better deciding method

Teachers Status

Display of the teacher status in order to see if you can assign him/her or not

Graphic Interface

Development of a graphical interface for building the DSD

Wish List

Tool to view each teacher's wish list, in which they indicate their teaching preferences
